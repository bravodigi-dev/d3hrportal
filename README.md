# Daythree Employee Knowledge Portal – Version 3

A GitHub Pages-ready HR portal based on the premium Daythree design direction.

## Included
- Official Daythree logo
- Premium purple and pink corporate design
- Responsive navigation
- Live HR knowledge search
- Quick access dashboard
- Employee information cards
- Leave, dress code, payroll, medical, overtime and grievance content
- FAQ filtering
- Dark mode
- Image preview modal
- Built-in Ask HR chat assistant
- No employee-specific names

## Upload to GitHub
Upload all files and the complete `assets` folder into the root of the repository.

Expected structure:

```
index.html
style.css
script.js
README.md
assets/
```

Then open **Settings → Pages**, choose **Deploy from a branch**, select **main** and **/(root)**.

## Chat Assistant
The included chat assistant works directly in the browser and does not require a backend. It provides answers from the built-in HR knowledge base.

For a future AI or ElevenLabs integration, connect through a secure backend. Do not publish private API keys inside GitHub Pages files.


## Content Coverage
This version includes all information and FAQ topics from the original Kilimanjaro Employee Knowledge Portal checked on 22 July 2026, including:

- Dress code

- Grievance and whistleblowing
